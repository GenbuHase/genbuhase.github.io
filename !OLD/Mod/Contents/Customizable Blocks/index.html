Customizable Blocks

1. Overview

このModでは、自由自在にカスタマイズできるブロックを追加します。

2. Detail

このModを適用した状態でワールドに入ると、現在のワールドのフォルダにCustomizableBlocks.cfgが生成されます。
CustomizableBlocks.cfgを、必ず以下の形式に沿って編集してください。

入力された内容に応じて追加されるブロックの情報が変化します。 

[
	{
		"Damage": 対応ダメージ値(int型),

		"Property": {
			"Textures": [ //ブロックテクスチャ設定
				[テクスチャネーム(String型), テクスチャダメージ値(int型)],
				[テクスチャネーム(String型), テクスチャダメージ値(int型)],
				...
			],

			"Shapes": [ //ブロック形状設定
				[開始X(int型), 開始Y(int型), 開始Z(int型)],
				[終了X(int型), 終了Y(int型), 終了Z(int型)]
			]
		}
	},

	{
		...
	},

	...
]
